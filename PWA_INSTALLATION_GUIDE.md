## 📱 Android Installation

### Method 1: Chrome Browser (Recommended)
1. Open **Chrome** on your Android device
2. Navigate to: `https://farm-water-tracker-1.preview.emergentagent.com`
3. Tap the **3-dot menu** (⋮) in the top right
4. Select **"Add to Home screen"** or **"Install app"**
5. Confirm the installation
6. The app icon will appear on your home screen

### Method 2: In-App Install Prompt
1. Visit the app URL in Chrome
2. An install banner will automatically appear at the bottom
3. Tap **"Install"** on the banner
4. Done! The app is now on your home screen

### Features on Android:
✅ Full-screen app experience (no browser UI)
✅ Home screen icon
✅ Splash screen on launch
✅ Works offline
✅ Push notifications (when enabled)
✅ Runs independently like a native app

---

## 🍎 iOS Installation (iPhone/iPad)

### Safari Browser Only
1. Open **Safari** on your iOS device (Chrome doesn't support PWA install on iOS)
2. Navigate to: `https://farm-water-tracker-1.preview.emergentagent.com`
3. Tap the **Share button** (□↑) at the bottom
4. Scroll down and tap **"Add to Home Screen"**
5. Edit the name if desired
6. Tap **"Add"** in the top right
7. The app icon will appear on your home screen

### Features on iOS:
✅ Standalone app mode
✅ Home screen icon
✅ Splash screen
✅ Works like a native app
✅ No browser chrome

**Note:** iOS has some limitations compared to Android - offline support is limited and push notifications are not available.

## ❓ Troubleshooting

### Install option not showing?
- **Android:** Make sure you're using Chrome
- **iOS:** Use Safari browser only
- **Desktop:** Try Chrome, Edge, or Brave
- Clear browser cache and try again

### App not working offline?
- Visit the app while online first to cache resources
- Check if service worker is registered (Dev Tools → Application → Service Workers)

### Icon not appearing correctly?
- Clear browser cache
- Uninstall and reinstall the app
